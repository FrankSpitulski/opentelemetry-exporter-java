# Lightstep Example

## Configuration

Update config.properties with correct values for 
- _lightstep.access.token_
- _lightstep.collector.host_
- _lightstep.collector.port_
- _lightstep.collector.protocol_
- _lightstep.service.name_

## Build
```shell script
make buid
```

## Run
```shell script
make run
```
